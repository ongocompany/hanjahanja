- **Manifest key**: 이미 `apps/extension/wxt.config.ts`에 추가됨

---

## 1단계: Google Cloud Console - 항목 아이디 입력

Google Cloud Console에서 **항목 아이디**에 입력:

```
ckncdpmdnfacolnjmnhmckbpjojmfjom
```

## 2단계: OAuth 클라이언트 ID 생성

1. **사용자 인증 정보** → **OAuth 클라이언트 ID 만들기**
2. 애플리케이션 유형: **Chrome 앱**
3. 애플리케이션 ID: `ckncdpmdnfacolnjmnhmckbpjojmfjom`
4. 생성하면 **클라이언트 ID** (`xxxx.apps.googleusercontent.com`) 받게 됨

## 3단계: Supabase Auth에 Google 프로바이더 설정

1. Supabase 대시보드 → **Authentication** → **Providers** → **Google**
2. Google 클라이언트 ID + 클라이언트 시크릿 입력
3. Redirect URL 확인 (Supabase가 제공하는 콜백 URL)

## 4단계: 크롬 확장에서 인증 구현

`chrome.identity.launchWebAuthFlow()` 사용해서 OAuth 플로우 연결
